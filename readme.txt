Welcome to use Buffalo!

[How to start]

* If you downloaded the buffalo-<version>-bin.zip, just unzip and copy buffalo-demo.war to
tomcat/webapps, start tomcat, input the url below in your browser: 

http://localhost:8080/buffalo-demo

See the samples at the first eye. 

and, the "site" sub folder contains the documentation.

* If you download the buffalo-<version>-src.zip, you need maven(http://maven.apache.org) to build it.
